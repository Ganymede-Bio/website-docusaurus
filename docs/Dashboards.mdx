The <div className="button darkblue_button"><AreaChartOutlined /> **Dashboard**</div> page offers charts and dashboard visualizations based on data stored within Ganymede.  

- Select Charts to create a chart based on the library of chart types
  - There are several pre-configured datasets based on run metadata
  - If you need access to additional datasets, reach out to your Ganymede Implementation Manager

- Select Dashboards to create a dashboard composed of the charts
