The <div className="button border_button"><HomeOutlined /> **Home**</div> page contains links to available _flows_ in an environment.  Clicking on any listed _flow_ brings you to the <div className="button border_button"><HomeOutlined /> **Flow View**</div> page, from which the user can run the selected _flow_ and view historical runs.

![Ganymede App Flow Editor Homepage](https://ganymede-bio.mo.cloudinary.net/apiServer/HomeScreen_20221220.png)

The search bar at the top filters available _flows_ by name.

## Navigation Sidebar

### Top Section

The top part of the sidebar contains pages for building and running flows, as well as observing data processed by _flows_.

[<div className="button border_button"><HomeOutlined /> **Home**</div>](./Home.mdx) - Home screen used to access _flows_ to run

[<div className="button border_button"><NodeIndexOutlined /> **Flow Editor**</div>](./FlowEditor.mdx) - Editor for modifying _flows_ and their underlying code

[<div className="button border_button"><CheckCircleOutlined /> **Flow Runs**</div>](./FlowRuns.mdx) - Table for tracking status of current and prior _flow_ runs

[<div className="button border_button"><FileSearchOutlined /> **Data Explorer**</div>](./DataExplorer.mdx) - SQL editor for querying tabular data created within _flows_

[<div className="button border_button"><FolderOutlined /> **Files**</div>](./Files.mdx) - File browser for accessing snapshotted input and output data processed through _flows_

### Bottom Section

The bottom part of the sidebar contains documentation and pages for administering access to Ganymede.

[<div className="button border_button"><NotificationOutlined /> **Notifications**</div>](./Notifications.mdx) - Activity (e.g. - _flow_ saves, _flow_ runs, etc.) associated with Ganymede tenant environment

[<div className="button border_button"><FileTextOutlined /> **Docs**</div>](https://docs.ganymede.bio) - Link to Ganymede's documentation, which can also be accessed directly at [docs.ganymede.bio](https://docs.ganymede.bio/) 

[<div className="button border_button"><SubnodeOutlined /> **Admin**</div>](./AdminControls.md) - Panel for configuring levels of access for different individuals

[<div className="button border_button"><UserOutlined /> **Log Out**</div>](https://www.ganymede.bio) - Log out from Ganymede cloud
