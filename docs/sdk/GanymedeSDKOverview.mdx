The Ganymede SDK provides a methods for interacting with Ganymede from editor and analysis notebooks.

```python
from ganymede_sdk import Ganymede
```

- The [**Ganymede**](./GanymedeClass.mdx) Class contains methods for listing and retrieving data stored in Ganymede
- [**Context**](./ModuleEditor.mdx) objects contain methods for accessing run metadata
- [**GPT**](./GPT.mdx) is a class found in Ganymede SDK that provides coding assistance for use in Editor notebooks.  Usage requires configuring OpenAI API Key and Organization in your environment.

:::tip

Functions can be introspected by typing ? or ?? before the function name in a notebook cell.  For example:

```python
from ganymede_sdk import Ganymede

g = Ganymede()

# shows function signature and docstring 
?g.retrieve_sql
```
