The Ganymede SDK provides a set of convenience functions for interacting with Ganymede from editor and analysis notebooks.

```python
import ganymede_sdk.editor
import ganymede_sdk.io
```

- [**ganymede_sdk.editor**](./ModuleEditor.mdx) provides the GanymedeContext, used to store runtime data, as well as functions for testing and validating flows
- [**ganymede_sdk.io**](./ModuleIO.mdx) contains functions for listing and retrieving data stored in Ganymede
- [**GPT**](./GPT.mdx) is a class found in Ganymede SDK that provides coding assistance for use in Editor notebooks.  Usage requires configuring OpenAI API Key and Organization in your environment.

:::tip

Functions can be introspected by typing ? or ?? before the function name in a notebook cell.  For example:

```python
from ganymede_sdk.io import retrieve_sql

# shows function signature and docstring 
?retrieve_sql
```
