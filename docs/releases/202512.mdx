### December 19 2025
**Improvements**
- **API**: Added support for listing and retrieving Instruments from the API

**Bug Fixes**
- **Flows**: Fixed an issue where Flow limit on Max Concurrent Runs was not being enforced correctly

**Agent Version**: 5.5.96

**NOTE:** This is the last release for 2025. The next planned release will be January 8th 2026. Happy Holidays from all of us!

### December 11 2025
**Improvements**
- **Web UI**: Improved selection behavior when creating Runs for Processes, Unit Operations and Instruments
- **Flows**: Increased robustness for Flow Runs against transient network issues

**Agent Version**: 5.5.96

### December 4 2025
**Improvements**
- **Flows**: Increased robustness for Flow Runs launched under high event load
- **Agents**: The latest Agent version has improved performance when applying multiple Tags to a File

**Bug Fixes**
- **Web UI**: Fixed an issues where Flow Run logs would fail to be displayed[]
- **Web UI**: Fixed an issue where the cancel button was ignored when editing Processes
