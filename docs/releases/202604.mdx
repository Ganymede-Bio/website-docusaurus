### April 2 2026

**Features**
- **Agents**: Added support to allow agents to capture both timezone information &  clock drift from host PCs.
    - The amount of clock drift and timezone are now captured and stored alongside host inferred data, making timing discrepancies 
  observable and correctable


**Improvements**
- **Agents**: Fixed an issue where some Gateway Agent EXEs were consuming more memory than expected, leading to performance degradation over time
- **Web UI**: Bulk metadata tag update - users with "Writer" environment permissions now have the ability to update file metadata tags in bulk through the UI

**Bug Fixes**
- **Web UI**: Fixed an issue with page scroll behavior on the Data Explorer and Modular Analysis pages

**Gateway Version**: 5.6.29
