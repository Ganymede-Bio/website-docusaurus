### July 16 2026

**Bug Fixes**
- **Web UI / Flow Runs**: Fixed an issue where the chart on the Flow Runs page was cut off on the right and bottom edges with no usable scrollbars, making some chart data unreachable. The chart now resizes to fit the available width and the page scrolls so the full chart is always visible.

**Gateway Version**: 5.6.55

### July 02 2026

> Note that there will be no release next week due to the Ganymede holiday break.

**Features**
- **Web UI / Localization**: The Ganymede web app is now available in 12 additional languages, selectable from a new language picker.
    - Supported languages now include Spanish, Japanese, German, French, Italian, Korean, Dutch, Portuguese, Swedish, Danish, Croatian, and Chinese (Simplified), in addition to English.

![Language picker with supported languages](/img/20260702_language_picker.png)

- **Web UI / Integrations**: For Tempo MES-connected Ganymede environments, a new Connection Health table on the Integration page displays the latest sync status for Flow-based Tempo data connections.
    - Each row surfaces the connection status, last Flow run time, data tables successfully or unsuccessfully synced during the Flow run, and the number of new rows written to each table.

![Connection Health table on the Integration page](/img/20260702_connection_health.png)

**Improvements**
- **Web UI / Left-Hand Navigation**: The left-hand navigation has been redesigned with a new grouped layout, organizing related platform functionality into collapsible sections. The left-hand navigation bar will auto-expand upon selections and can be re-collapsed via the collapse button at the bottom of the navigation bar. Please note that several navigation destinations have also been renamed for clarity.
    - Home (unchanged)
    - Processes
        - Runs
        - Processes
        - Unit Operations
        - Instrument Types
        - Instruments
    - Connectors (previously "Gateway")
        - Connections
        - Hosts
        - Agent Templates
    - Flows
        - Flow Editor
        - Flow Runs
    - Data
        - Data Explorer
        - File Browser (previously "Files")
    - Analysis
        - Modular Analysis
        - Dashboard
        - Virtualization (previously "Sessions")

![Redesigned left-hand navigation with grouped, collapsible sections](/img/20260702_lhn_redesign.gif)

**Bug Fixes**
- **Web UI / Process Runs**: Fixed an issue where the pagination controls and rows-per-page selector were unresponsive on the Files table within the Process Runs page.

**Gateway Version**: 5.6.53
