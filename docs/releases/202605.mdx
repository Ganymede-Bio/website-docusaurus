### April 9 2026

**Improvements**
- **Flows**: A number of performance improvements have been added to Flow Runs that use API keys to call the Ganymede public API. Previously, Flows containing nodes with API key-based calls could trigger intermittent 403 errors due to empty/blank credentials - these improvements increase the resilience of credential lookups and significantly reduce the possibility of hitting GCP quota limits.

**Bug Fixes**
- **Web UI / Modular Analysis Applications**: Fixed an issue where Modular Analysis Applications were not rendering properly within the UI, causing them to be unusable. The impact of this bug was confined strictly to the display of the application in the web UI - all underlying application code, data, and configuration was unaffected. The fix has been thoroughly tested and confirmed stable.

**Gateway Version**: 5.6.24
