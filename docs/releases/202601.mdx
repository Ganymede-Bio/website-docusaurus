### January 22 2026
**Features**
- **Web UI**: Added support for archiving files
  - Files can be archived from the Files page or via the API
  - Archived files are hidden from default views but can be accessed via filters
  - Archived files can be restored

**Bug Fixes**
- **Web UI**: Fixed an issue where bulk downloads of multiple Files could fail when some Files were very large
- **Agent**: Fixed an issue where new installs would fail to start

**Agent Version**: 5.5.102

### January 15 2026
**Features**
- **Web UI**: Added support for mapping many Instrument Runs to Unit Operations and vice versa

**Bug Fixes**
- **Web UI**: Fixed an issue where manual upload of a File containing special characters in its name would fail to download
- **Agent**: Fixed an issue where Linux agents could fail to update

**Agent Version**: 5.5.100

### January 8 2026
**Features**
- **Web UI**: Added fields to Instrument objects to track latitude, longitude, timezone, and the start time of observation
- **API**: Added support for data paging to multiple API endpoints to improve performance when handling large datasets. These endpoints are reachable via `v2` routes and should be used in place of the 
    original endpoints which are supported, but deprecated.

**Improvements**
- **Agent**: Improved startup performance when checking for existing message subscriptions
- **Web UI**: Tables will now be listed in alphabetical order by default
- **Web UI**: Connections will now display details related to their associated Host

**Bug Fixes**
- **API**: Fixed an issue where paged requests to list Tables would not return the correct total count of items

**Agent Version**: 5.5.99
