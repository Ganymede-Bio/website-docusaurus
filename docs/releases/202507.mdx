### July 3 2025

**Features** 
- **API**: Library files can now be committed using the public API
    - This allows for function reuse between Flows
    - Add these files in a folder named `lib` in your Flow repository
    - These files will be automatically included in the Flow when it is executed and are accessible within Notebook editors

**Improvements** 
- **Modular Analysis**: Improved handling of session end time tracking 
- **Modular Analysis**: Added support for retrieving the current session User

**Bug Fixes**
- **Agents**: Improved the startup process for Agents to ensure they are ready before the Windows Service Manager closes them

**Agent Version**: 5.5.41
