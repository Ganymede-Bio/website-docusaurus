### June 11 2026

**Features**
- **Gateway Installer**: The Windows gateway installer is now available in 13 languages
    - A language selection dialog appears at the start of installation, letting users install the gateway in English, Danish, German, Spanish, French, Croatian, Italian, Japanese, Korean, Dutch, Portuguese, Swedish, or Chinese Simplified.

![Gateway installer language selection dialog](/img/gateway_installer_languages.png)

**Gateway Version**: 5.6.46

### June 05 2026

**Features**
- **Dashboards**: Dashboards now support role-based access with three new user roles. Each role is scoped to a specific environment, so users only see the dashboards and data for the environments they are permitted to access.
    - **Writer** - full content and data access, including SQL Lab, datasets, and charts.
    - **Runner** - view and run dashboards, charts, and datasets (no create or edit).
    - **Reader** - view-only access to published dashboards.

**Bug Fixes**
- **Agents**: Fixed an issue where email notifications would not be sent for connections in a failing state (for example, a file watcher pointing at a missing directory).
- **Web UI / Agent Templates**: Fixed an issue with the position of the Agent Template download button on the Agent Templates page. The download button has been moved to the page header row alongside the Code, View, and Edit actions.

**Gateway Version**: 5.6.44
