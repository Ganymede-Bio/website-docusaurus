### June 05 2026

**Features**
- **Dashboards**: Dashboards now support role-based access with three new user roles. Each role is scoped to a specific environment, so users only see the dashboards and data for the environments they are permitted to access.
    - **Writer** - full content and data access, including SQL Lab, datasets, and charts.
    - **Runner** - view and run dashboards, charts, and datasets (no create or edit).
    - **Reader** - view-only access to published dashboards.

**Improvements**
- **Web UI / Navigation**: The left-hand navigation has been redesigned. New collapsible/expandable groups organize related platform functionality together, and several labels have been renamed for clarity. The navigation rail will automatically expand upon selections and can be collapsed to an icon-only view to free up screen space.
    - **Grouped navigation** - related platform functionality is now organized into expandable, collapsible groups: Processes, Connectors, Flows, Data, and Analysis.
        - Processes: Runs, Processes, Unit Operations, Instrument Types, Instruments
        - Connectors: Connections, Hosts, Agent Templates
        - Flows: Flow Editor, Flow Runs
        - Data: Data Explorer, File Browser
        - Analysis: Modular Analysis, Dashboard, Virtualization
    - **Clearer labels** - several sidebar items were renamed for clarity:
        - "Gateway" renamed to "Connectors"
        - "Files" renamed to "File Browser"
        - "Sessions" renamed to "Virtualization"

**Bug Fixes**
- **Agents**: Fixed an issue where email notifications would not be sent for connections in a failing state (for example, a file watcher pointing at a missing directory).
- **Web UI / Agent Templates**: Fixed an issue with the position of the Agent Template download button on the Agent Templates page. The download button has been moved to the page header row alongside the Code, View, and Edit actions.

**Gateway Version**: 5.6.44
