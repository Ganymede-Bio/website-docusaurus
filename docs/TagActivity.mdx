The Tag Activity page provides a detailed view of all activities related to a single  [File Tag](./connectivity/Tags.mdx). All activities for the selected tag are displayed on a timeline in chronological order, with each activity showing its type and detailed information.

![Tag activity](https://ganymede-bio.mo.cloudinary.net/apiServer/TagActivity_View.png)

### Viewing Tag activity

To view a Tag's activity, click on the File Tag chip on the <div className="button darkblue_button"><FolderOutlined /> **Files**</div> page, or click on its file name. 

![File chip](https://ganymede-bio.mo.cloudinary.net/apiServer/FileTag_Chip.png)

### Reading Tag activity

Tag activity is grouped into the following categories:

- Files used in <div className="button darkblue_button"><CheckCircleOutlined /> **Flow Run**</div>. Details shown include time the file was used in the Flow run, link to the corresponding Flow run details, Flow run status, user that ran the Flow, and a summary of input and output files.
- Input and output files tagged in a <div className="button darkblue_button"><AppstoreOutlined /> **Virtualization Session**</div>. Details shown include application used (e.g. FlowJo), user that started the session, and a summary of input and output files.
- Files used in <div className="button darkblue_button"><ApiOutlined /> **Agent Connections**</div>. Details shown include connection name, link to Connection details, user that initiated the Connection, and a summary of files used in the [Agent Connection](./connectivity/Agent).
