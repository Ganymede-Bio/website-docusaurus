## NavBars

### Collapsible Sidebar

The navigation sidebar contains pages for building and running flows, as well as observing data processed by flows.

[<div className="button darkblue_button"><HomeOutlined /> **Home**</div>](./Home.mdx) - Home screen used to access flows to run

[<div className="button darkblue_button"><BranchesOutlined /> **Flow Editor**</div>](./flows/FlowEditor.mdx) - Editor for modifying flows and their underlying code

[<div className="button darkblue_button"><CheckCircleOutlined /> **Flow Runs**</div>](./flows/FlowRuns.mdx) - Table for tracking status of current and prior _flow_ runs

[<div className="button darkblue_button"><FileSearchOutlined /> **Data Explorer**</div>](./tables/DataExplorer.mdx) - Tools for querying and managing tabular data created by flows

[<div className="button darkblue_button"><FolderOutlined /> **Files**</div>](./files/FileBrowser) - File browser for accessing snapshotted input and output data processed through flows

### Environment Selector

This dropdown decides which environment that is currently being looked at. All tables, flows, and dashboards that are shown are associated with this environment. They can be color coded and have icons to distinguish them a bit easier.
In order to change them please feel free to contact the ganymede email in order to get it adjusted. 

### Header Navigation

The header contains buttons for accessing environment notifications, app documentation, and settings for administering access to Ganymede and secrets stored used on Ganymede platform.

[<div className="button darkblue_button"><BellOutlined /> **Notifications**</div>](./flows/Notifications.mdx) - Activity (e.g. - _flow_ saves, _flow_ runs, etc.) associated with Ganymede tenant environment

[<div className="button darkblue_button"><FileTextOutlined /> **Docs**</div>](https://docs.ganymede.bio) - Link to Ganymede's documentation, which can also be accessed directly at [docs.ganymede.bio](https://docs.ganymede.bio/) 

[<div className="button darkblue_button"><SettingOutlined /> **Admin**</div>](./admin/AdminControls.mdx) - Panel for configuring access to Ganymede and secrets for accessing third-party applications from Ganymede

[<div className="button darkblue_button"><UserOutlined /> **Log Out**</div>](https://www.ganymede.bio) - Log out from Ganymede cloud
