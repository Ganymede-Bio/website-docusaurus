## Overview

<img height="400" alt="Shared Computer Auto Logout" src="https://ganymede-bio.mo.cloudinary.net/apiServer/shared_computer_auto_logout.png" />

Before accessing Ganymede, users can select the "This is a shared computer" option, which will automatically log them out upon closing the application.

We recommend enabling this feature for shared lab workstations to ensure that users are securely signed out without needing to manually log off after their session.
